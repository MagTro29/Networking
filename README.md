# Networking Labs

This repository contains hands-on networking and network security labs
completed as part of my Bachelor of Information Technology
(Networking & Cybersecurity) at the University of Technology Sydney (UTS).

The work focuses on routing, switching, core network services, and
network security concepts commonly used in enterprise environments.

## Areas Covered

### Routing & Switching
- Static and dynamic routing (OSPF, EIGRP)
- VLANs and inter-VLAN routing
- Subnetting and IP addressing
- Network design and verification

### Network Security
- Firewall and ACL configuration
- VPN concepts (IPSec)
- AAA (RADIUS / TACACS+)
- Secure device management (SSH, HTTPS)

### Network Services
- DHCP configuration and troubleshooting
- DNS fundamentals
- Core enterprise network services

### Troubleshooting
- Connectivity verification
- Packet capture and analysis using Wireshark
- Log-based fault isolation

## Tools & Platforms
- Cisco Packet Tracer
- Cisco ASA (CLI)
- Wireshark
- Linux

## Disclaimer
All labs were performed in controlled academic environments.
No real credentials, production systems, or sensitive data are included.
